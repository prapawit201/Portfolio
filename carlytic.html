Senior Project : Calytics ( 2020 )

" The purpose of Carlytics was to develop a system for managing the driving behavior to focus on monitoring, managing, and controlling the driving behavior of drivers in the enterprise to be as efficient as possible. This system was developed as a web application for detecting the speed and engine revs of each enterprise vehicle. "



Infrastructure


 Infrastructure :
Web Server: Amazon EC2, Google Cloud Platform, Heroku
Web Service: NGINX
Operating System: Ubuntu
Database Server: Amazon RDS, MongoDB Atlas
Hardware: OBD-II Adapter, Android Smartphone
Service: Node-red


Tools and Techniques


  Tools :
Development: Visual Studio Code, Postman, Github
Design: Figma
Collaboration: Notion
Deployment: Docker

  Technology :
Front-end: React, Bootstrap 4, React Google Map, ChartJS, CanvasJS
Back-end: Node.js, Express.js
Authentication: JWT Authentication
Database: mySql ,MongoDB
